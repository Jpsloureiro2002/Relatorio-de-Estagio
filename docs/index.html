Sistema de Apoio à Administração

Por João Loureiro

Introdução:

O projeto visa o desenvolvimento de uma aplicação híbrida de gestão de tempo de trabalho para contabilização de horas, resultando na diminuição do tempo dedicado por cada colaborador a estas tarefas. A aplicação também contém funções adicionais, como a validação de certificados tais como o Cartão de Cidadão, Carta de Condução, International Bank Account Number (IBAN) e Passaporte, assim como a submissão de despesas de projeto.

O projeto foi desenvolvido com o uso das linguagens de programação Python para o primeiro prototipo e Flutter para a aplicação final.
